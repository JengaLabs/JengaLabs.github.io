• Rotten Carrots

    Book: "Learn CSS In One Day and Learn It Well CSS for Beginners with Hands-On Project The only book you need to start coding in CSS immediately"
    By: Jamie Chan

  • Ratings : 4/10

    While I enjoyed the book for a quick read, throughout the read it felt like examples of use cases were not really covered. The exercise files seemed to just be regurgitations of what’s discussed in a chapter. Each chapter just stating how to use a css property, not giving the reader a chance to show they really absorbed the material. Lack of explanation could be justified by the writer attempting to keep the book short and concise. That does not excuse exercise files lacking in any substance and proper examples to follow.

  • Book Summary

    A short 120 pages going into HTML elements and their respective CSS properties. Including exercise files that represent the content discussed in chapters. Learn CSS in One Day is a very surface level overview that could be created with a few Youtube Short clips with the same quality.

  • Chapter 1

    Two pages briefly covering what web languages are and tools to get started. Giving a brief explanation of opening the exercise files included.

  • Chapter 2

    Covering HTML elements and placements to create a site. While going over some meta and style elements specifics.Linking to Safari documentation regarding the iOS viewport.

  • Chapter 3

    Inline, internal and external styles the order of precedence. The purpose of classes and IDs to elements and how to select elements with pseudo-classes/elements. Finally the inconsistency of how each browser reads CSS.

  • Chapter 4

    CSS Box Model the most important aspect to begin understanding CSS. However the book moves past quickly to go over the CSS syntax regarding borders and colors. .

  • Chapter 5

    Positioning another important property to understand with good explanations of each. As well showing how z-index can layer elements. Missing an important chance to display use cases of these properties. Instead covering the float property in the same chapter.

  • Chapter 6

    Display and Visibility I found to be a very useful chapter. While I was familiar with the display property, the specifics behind inline and block values can be tricky. The Visibility property can be very powerful, however the true power behind it isn’t touched.

  • Chapter 7

    Background property leads to the creation of more stylistic sites. Background attachment being another unknown property to me. Again, no real discussion of how this property could be used effectively. Just notifying you that it indeed is a CSS feature.

  • Chapter 8

    Text and font properties can make or break a site. This chapter does cover changing site fonts, but not adding in your own custom fonts. While not exactly something a beginner would want to do, it’s something they should do.

  • Chapter 9

    Navigation Bars are one of the first steps to creating multi page sites. While there are many solutions to creating a navigation bar. The book only covers hyperlink styling and types of lists with their CSS properties. The exercise file does show one solution, but lacks explanation behind the solution.

  • Chapter 10

    Tables are a great introduction to CSS for beginners as it’s a concept many are already familiar with. As well soon enough the reader will learn its limitations leading them to the use of Flex boxes and more advanced practices.

  • Bonus Project

    A good site that uses the same tools taught throughout the book. Ideally the book would have had you built this site throughout the book. Finishing with the user having the same result and asked to create something of their own. Now armed with the knowledge and steps taken to create the site more directly.
